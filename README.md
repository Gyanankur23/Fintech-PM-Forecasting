## 📖 Overview
This project was developed as part of the **Microsoft Elevate AICTE Internship (Power BI track)**.  
It focuses on **Fintech Project Management**, transforming scattered project data into actionable insights using **Power BI dashboards** and **predictive analytics**.

Key highlights:
- 📊 Budget vs. Actual Analysis  
- ⚡ Risk‑level completion forecasting  
- 👥 Employee & Department performance metrics  
- 🌍 Regional insights with hourly rates and budget variances  
- 🤖 Predictive modeling (ARIMA/LSTM) for project completion and resource demand
